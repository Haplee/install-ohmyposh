Oh My Posh Installers

Scripts precisos y automatizados para configurar y embellecer tu terminal en segundos. Soporte nativo para Linux y Windows.

Instalar en Linux Instalar en Windows

⚡ Todo en Uno

Descarga Oh My Posh, inyecta la configuración en tu shell y descarga las fuentes Meslo Nerd Font necesarias.

🐧 Inteligente en Linux

Detecta automáticamente Bash o Zsh y añade las rutas correctas sin duplicar configuraciones.

🪟 Nativo en Windows

Interactúa con UAC para pedir permisos Administrator y muestra un tracker visual (GUI) en Python.

Linux (Bash / Zsh)

Clona el repositorio e instala directamente desde tu consola:

git clone https://github.com/Haplee/install-ohmyposh.git cd install-ohmyposh/linux chmod +x install_linux.sh ./install_linux.sh

Tip: Tras instalar, no olvides lanzar source ~/.bashrc o reiniciar tu terminal.

Windows (PowerShell)

Usa la versión compilada como ejecutable o compílala tú mismo fácilmente utilizando Python.

cd windows && .\compilar_windows.bat

Asegúrate de ejecutar el resultado generado final windows\dist\OhMyPosh_Installer.exe. Pedirá permisos de Administrador para inyectar correctamente las fuentes MesloLGS NF.
